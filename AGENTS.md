## Development

When starting the dev server, use background mode:

```
astro dev --background
```

Manage the background server with `astro dev stop`, `astro dev status`, and `astro dev logs`.

## Design System — Acero Frío (v3)

Agencia técnica de resolución de problemas. La identidad es sobria, industrial y competente — ni cálida/artesanal (terracota) ni genérica (navy+mint).

### Paleta
- **Base**: `#0C0D0E` (casi-negro neutro-frío) — industrial, técnico
- **Surface**: `#151618` / `#1E1F22` — capas de profundidad
- **Primary (acero)**: `#8BA0C0` — acero azul-gris, frío, profesional. Solo en CTAs e interactivos
- **Secondary**: `#6A7B8A` — gris técnico para badges y etiquetas
- **Texto**: `#F0F0F2` principal, `#A0A2A8` secundario, `#686A72` terciario
- **80% fondo, 15% texto, 5% color**

### Tipografía
- **Fraunces** (serif variable, Google Fonts) para headings
- **Sora** (sans redondeada) para body
- **JetBrains Mono** para mono

### Filosofía
- **Problema → Solución**: Todo el copy vende soluciones a problemas, no software
- **Sin señales de template**: No hay badge `./disponible.sh`, grid pattern, ping dot, glow CTA, gradient text, glass nav, ni violeta
- **Layout**: Hero sin mockup, Portfolio antes que Services, section-divider con ◆ mark, animación clip-path
